# Banking System Project

A simple menu-driven Banking System developed using Python.  
This project demonstrates core programming fundamentals through real-world banking operations.

---

## Features
- Create Account
- Deposit Money
- Withdraw Money
- Check Balance
- Transaction History
- Search Transactions
- Sort Transactions
- Interest Calculation

---

## Concepts Covered
- Variables & Data Types
- Conditional Statements
- Loops & Nested Loops
- Functions
- Lists & Strings
- Searching & Sorting
- Object-Oriented Programming Basics
- Exception Handling

---

## Technologies Used
- Python
